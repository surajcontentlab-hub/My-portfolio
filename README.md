## ✨ Features

- ⚡ **Particle Network Animation** — Interactive canvas background in hero section
- ✍️ **Typing Animation** — Rotating text showcasing my skills
- 📱 **Fully Responsive** — Works on mobile, tablet & desktop
- 🎯 **Portfolio Filter** — Filter projects by category (Video, Web, AI, Design)
- 📊 **Animated Counters** — Stats that count up when scrolled into view
- 🔢 **Skill Bars** — Animated progress bars for each skill
- 🌙 **Smooth Scroll** — Seamless navigation between sections
- 💬 **WhatsApp Integration** — Direct chat button for instant contact
- 🎨 **Modern UI** — Clean light theme with orange & navy color palette

## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/surajcontentlab-hub/portfolio.git
   ```

2. **Open the folder**
   ```bash
   cd portfolio
   ```

3. **Open in browser**
   - Just double-click `index.html`  
   - OR open with Live Server in VS Code
